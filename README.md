# 7350
Cornell's MATH 7350: Non-Archimedean and Tropical Geometry, taught by Farbod Shokrieh

# Conventions
Write 
* `\b?` for `\mathbf{?}`
* `\c?` for `\mathcal{?}`
* `\f?` for `\mathfrak{?}`
* `\s?` for `\mathscr{?}`

The Berkovich spectrum of a ring is `\Berkspec`. See the file `7350.sty` for other macros.
